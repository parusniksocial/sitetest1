# Getting Started
Installing Parusnik:
## Installing
1. Download Sitetest1
2. Unpack zip-archive
3. Edit MYSQL settings in DefaultSettings.php
4. Now you installed Parusnik
## Customizing

1. Edit customization settings in DefaultSettings.php
